# Job Portal Application

Ek full-stack web application jahan users job opportunities dekh sakte hain aur nayi jobs post kar sakte hain.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB

## Key Features
- **Job Listing:** Saari available jobs ko list karna.
- **Job Posting:** Nayi job entry add karna.
- **Job Removal:** Kisi bhi job ko delete karne ka feature.
- **Responsive UI:** Mobile aur Desktop dono par sahi dikhta hai.
